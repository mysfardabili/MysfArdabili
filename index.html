About

Halo, saya M. Yusuf Ardabili, seorang Mahasiswa Sistem Informasi Universitas Ahmad Dahlan yang berfokus pada pengembangan Web Application dengan pendekatan Fullstack. Memiliki pengalaman membangun aplikasi menggunakan React.js dan Next.js yang terintegrasi dengan REST API, serta memahami alur data antara frontend dan backend.

Saat ini sedang mendalami pengembangan backend, termasuk pembuatan API sederhana dan pengelolaan database untuk mendukung kebutuhan aplikasi. Terbiasa mengembangkan sistem dengan konsep multi-role (admin dan user) serta memahami dasar logika bisnis dalam aplikasi berbasis web.

Proaktif dalam pengembangan diri dengan membangun proyek mandiri di luar tugas kuliah. Mampu bekerja secara mandiri maupun dalam tim, serta berkomitmen pada penulisan kode yang rapi, terstruktur, dan mudah dikembangkan.
